# Computer Vision Finger Counter

## Description

This project is a real-time finger counting application developed with Python, OpenCV and MediaPipe.

The application detects one or more hands using a webcam, extracts hand landmarks and determines whether each finger is open or closed. The total number of open fingers is displayed on the screen in real time.

## Technologies

- Python
- OpenCV
- MediaPipe

## Features

- Real-time webcam processing
- Hand landmark detection
- Finger counting
- Live finger count display

## How to Run

```bash
pip install -r requirements.txt
python main.py
```
